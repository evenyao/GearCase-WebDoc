# 安装

# 本地引用

在 GitHub 仓库下载，或使用 git 命令克隆到本地，将 GearCase 重命名为 gear-case ，并将文件夹移动到项目目录 node_modules 内后注册使用。

git clone https://github.com/evenyao/GearCase.git

️⚠️ ️该方式可能会出现因缺少 node-sass 等依赖而报错的问题，您可以在下载之后使用 npm install 来解决该问题。所以我们推荐您使用 NPM / Yarn 的安装方式


# NPM / Yarn 安装

使用 NPM 或者 Yarn 方式进行安装，这里更加推荐使用该方式进行安装。

npm i -S gear-case

或

yarn add gear-case

如果您已经使用了上述任意一种方式进行了安装，请继续阅览快速上手章节。
