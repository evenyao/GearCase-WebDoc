# 版本信息

2018.12.6 发布 0.1.2 版
️☕️ ️该版本暂不包含 Vue 动画，简单动画效果将在下一个大版本进行添加。

2018.12.12 发布 0.1.4 版
️️修复了之前组件的一些样式问题，引了 scss。

2018.12.24 发布 0.1.5 版
️️新增 cascader 级联选择器组件。

2018.12.27 发布 0.2.1 版
️️新增 springs 伸展动画组件。得益于此，cascader、layout、collapse 等组件已经拥有了动画效果！

2018.12.29 发布 0.2.5 版
️️新增 slides 轮播幻灯片组件。

最新版本️

2018.12.31 发布 0.2.14 版
️️赶在 2019 新年到来之前升级了 NPM 包。现在可以正常使用 NPM 和 Yarn 方式进行库的使用了！并优化了整体线上代码仓库、更新了部分文档。
