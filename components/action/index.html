# 竖直方向

提供竖直方向展开收起动画。

基础用法

<template>
  <div>
    <gear-button class="control" @click="toggle">竖直 收缩/展开</gear-button>
    <gear-springs :visible="visible">
      <div class="demo"></div>
    </gear-springs>
  </div>
</template>

<script>
  export default {
    data() {
      return {
        visible: true
      }
    },
    methods: {
      toggle() {
        this.visible = ! this.visible
      }
    }
  }
</script>

Attributes

参数 说明 类型 可选值 默认值
visible 控制内容的展开与收起，动态绑定，为必选属性 Boolean - -
duration 动画过渡时间，以 ms 为单位 String / Number - 300


# 水平方向

提供水平方向展开收起动画。

基础用法

水平 收缩/展开 
<template>
  <div>
    <gear-button class="control" @click="toggle">水平 收缩/展开</gear-button>
    <gear-springs :visible="visible" horizontal duration="1000">
      <div class="demo"></div>
    </gear-springs>
  </div>
</template>

<script>
  export default {
    data() {
      return {
        visible: true
      }
    },
    methods: {
      toggle() {
        this.visible = ! this.visible
      }
    }
  }
</script>

Attributes

参数 说明 类型 可选值 默认值
visible 控制内容的展开与收起，动态绑定，为必选属性 Boolean - -
horizontal 决定动画横向展开与收起，不设置时默认为竖直方向 Boolean - false
duration 动画过渡时间，以 ms 为单位 String / Number - 300
