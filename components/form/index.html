基础用法 

<gear-input value="Normal"></gear-input>
<gear-input type="password" placeholder="type password"></gear-input>

<gear-input value="双向数据绑定" v-model="message"></gear-input><p class="message">{{message}}</p>

Attributes

参数 说明 类型 可选值 默认值
value 原生属性 String - -
placeholder 原生属性 String - -
type 原生属性 String - -
disabled disabled 状态 无法被选中（点击） Boolean - false
readonly readonly 状态 可以被选中（点击） Boolean - false
error error 状态 String - -

# 动态获取

使用 ajax 从后端动态获取 source 数据。每次选择都会进行请求，并进行 loading 提示。

⚠️ ️使用动态获取的方式建议在选择器容量非常大的时候再进行使用。因需严格统一后端传递来的 JSON 数据格式，较为繁琐。因此若数据量较少，推荐使用静态方式。

// ajax 请求
function ajax (parentId = 0) {
  return new Promise((success, fail) => {
    setTimeout(() => {
      let result = db.filter((item) => item.parent_id == parentId)
      result.forEach(node => {
        // 判断是否是最终节点
        if( db.filter((item) => item.parent_id === node.id).length > 0 ){
          node.isLeaf = false
        } else {
          node.isLeaf = true
        }
      })
      success(result)
    }, 300)
  })
}

export default {
  components: {
    'gear-icon': Icon,
    'gear-cascader': Cascader
  },
  data(){
    return {
      selected: [],
      source: [],
    }
  },
  created () {
    // ajax 动态获取
    ajax(0).then(result => {
      console.log(result)
      this.source = result
    })
  },
  // loadData 传入
  methods: {
    loadData({ id }, updateSource) {
      ajax(id).then(result => {
        updateSource(result)
      })
    },
  }  
}

popover-height 选择器弹出 popover 容器高度设置属性 String - 200px
:load-data 设置请求并传入到组件内部的函数 Function - -
