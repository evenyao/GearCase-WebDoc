# collapse 折叠面板

# 默认模式

点击多个标题可折叠展开多个内容

基础用法

标题1
内容1
标题2
内容2
标题3
内容3
选中（开启）的标题：
[ "1" ]
<gear-collapse :selected.sync="selected">
  <gear-collapse-item title="标题1" name="1">内容1</gear-collapse-item>
  <gear-collapse-item title="标题2" name="2">内容2</gear-collapse-item>
  <gear-collapse-item title="标题3" name="3">内容3</gear-collapse-item>
</gear-collapse>
<div style="padding: 20px 0;">选中（开启）的标题：
  <div class="selected">{{selected}}</div>
</div>


# single 模式

点击标题会折叠被点击标题之外的其他标题内容

基础用法

标题1
内容1
标题2
内容2
标题3
内容3
选中（开启）的标题：
[]
<gear-collapse :selected.sync="selected" single>
  <gear-collapse-item title="标题1" name="1">内容1</gear-collapse-item>
  <gear-collapse-item title="标题2" name="2">内容2</gear-collapse-item>
  <gear-collapse-item title="标题3" name="3">内容3</gear-collapse-item>
</gear-collapse>
<div style="padding: 20px 0;">选中（开启）的标题：
  <div class="selected">{{selected}}</div>
</div>

gear-collapse Attributes

参数 说明 类型 可选值 默认值
selected 当前展开 title 的 name 组成的数组，用 .sync 修饰符绑定 Array 1 ~ 标题总数 -
single 设置只能展开单个 title Boolean - true

gear-collapse-item Attributes

参数 说明 类型 可选值 默认值
title 标题内容 String - -
name title 所对应的 name 属性，唯一且必填。 String - -

gear-tabs Attributes

参数 说明 类型 可选值 默认值
selected 设置默认被选中的 tabs-item ，用 .sync 修饰符绑定 String 必须是 name 中含有的字符串 -

gear-tabs-nav Attributes

参数 说明 类型 可选值 默认值
actions 可以设置 template 标签，slot 属性取名为 actions，在内部放置 gear-button 用来触发某些事件 - - -

gear-tabs-item Attributes

参数 说明 类型 可选值 默认值
name 设置 name ，标记 tabs-item 名称，对应 tabs-pane 中设置的 name String / Number 同一组 name 与 tabs-pane 设置的 name 应该相同 -
disabled 设置 tabs-item 是否被禁用 Boolean - false

gear-tabs-pane Attributes

参数 说明 类型 可选值 默认值
name 设置 name ，标记 tabs-pane 名称，与 tabs-item 中设置的 name 对应 String / Number 同一组 name 与 tabs-item 设置的 name 应该相同 -

⚠️ ️若出现 tabs-item 与 tabs-pane 内容不匹配的情况，可能是由于 name 属性设置不对应造成的，请务必检查每一组 tabs-item 与 tabs-pane 的 name 属性必须相同。




# slides 轮播幻灯片

在有限空间内，循环播放同一类型的图片内容

基础用法

<gear-slides :slides="slides" :setWidth="setWidth"
             :rollAuto="rollAuto" :rollSpeed="rollSpeed"
>
</gear-slides>

// 传值 data 数据结构

data() {
  return {
    // 图片信息
    slides: [
      {
        clickUrl: "#1",    // 跳转 url 地址
        img: 'xxx/1.png',  // 图片地址
        desc: "1.png"      // alt 文本信息
      },
      {
        clickUrl: "#2",
        img: 'xxx/2.png',
        desc: "2.png"
      },
      {
        clickUrl: "#3",
        img: 'xxx/3.png',
        desc: "3.png"
      },
      {
        clickUrl: "#4",
        img: 'xxx/4.png',
        desc: "4.png"
      },
    ],
    // 设置轮播图宽度
    setWidth: 500,
    // 自动播放间隔 秒为单位
    rollAuto: 5,
    // 轮播滚动速度
    rollSpeed: 20
  }
},

Attributes

参数 说明 类型 可选值 默认值
slides 静态数据组成的数组，包含跳转链接，图片地址与描述 Array - -
setWidth 设置轮播图宽度，以 px 为单位 Number - 600
setHeight 设置轮播图高度，以 px 为单位。不设置时默认宽高比为 3:2 Number - setWidth / 1.5
rollAuto 自动播放间隔，以 s 为单位。不设置时默认不会自动轮播 Number - 3
rollSpeed 轮播滚动速度 Number - 20
selectedButton 设置底部选择按钮 Boolean - true

⚠️ ️基础用法 demo 中只设置了 setWidth, 因此高度按 3:2 自适应。若想按照其他比例设置高宽，请设置 setWidth 与 setHeight 两个属性。




# nav 导航菜单

# 横向导航

提供横向导航功能的菜单。

基础用法 

<gear-nav :selected.sync="selected">
  <gear-nav-item name="home">首页</gear-nav-item>
  <gear-nav-sub name="about">
    <template slot="title">关于</template>
    <gear-nav-item name="team">团队</gear-nav-item>
    <gear-nav-item name="project">项目</gear-nav-item>
    <gear-nav-sub name="contacts">
      <template slot="title">联系</template>
      <gear-nav-item name="phone">手机</gear-nav-item>
      <gear-nav-item name="github">Github</gear-nav-item>
      <gear-nav-sub name="social">
        <template slot="title">社交</template>
        <gear-nav-item name="wechat">微信</gear-nav-item>
        <gear-nav-item name="weibo">微博</gear-nav-item>
      </gear-nav-sub>
    </gear-nav-sub>
  </gear-nav-sub>
  <gear-nav-item name="me">我的</gear-nav-item>
</gear-nav>

  <gear-nav-sub name="about">
    <template slot="title">关于</template>
    <gear-nav-item name="team">团队</gear-nav-item>
    <gear-nav-item name="project">项目</gear-nav-item>
    <gear-nav-sub name="contacts">
      <template slot="title">联系</template>
      <gear-nav-item name="phone">手机</gear-nav-item>
      <gear-nav-item name="github">Github</gear-nav-item>
      <gear-nav-sub name="social">
        <template slot="title">社交</template>
        <gear-nav-item name="wechat">微信</gear-nav-item>
        <gear-nav-item name="weibo">微博</gear-nav-item>
      </gear-nav-sub>
    </gear-nav-sub>
  </gear-nav-sub>

Attributes

参数 说明 类型 可选值 默认值
selected 设置默认被选中的标题，用 .sync 修饰符绑定 String 必须是 name 中含有的字符串 -
vertical 设置竖直方向导航，不设置时默认为横向导航 Boolean - false
